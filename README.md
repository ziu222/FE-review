## Folder Structure
- `customer/` — Main customer-facing HTML files
- `asset/css/` — CSS styles (Tailwind, Bootstrap, custom)
- `asset/js/` — JavaScript for dynamic features
- `asset/img/` — Images and icons
- `components/` — Optional reusable HTML/CSS components

## Getting Started
1. Open `customer/home.html` in your browser to view the main page.
2. To customize styles, edit files in `asset/css/` and rebuild Tailwind if needed.
3. Product data and dynamic features are managed in `asset/js/main.js`.

## Notes
- Unused assets and components can be safely deleted for a cleaner project.
- For production, consider bundling and minifying CSS/JS.
